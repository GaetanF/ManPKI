# ManPKI
X.509 PKI API Manager

Setup
==========================================

- Fill /etc/manpki/manpki.conf file in manpki with content :
    [default]
    websecret = randomstring
    tokensecret = randomstring


Features
==========================================
- RESTFUL API
- X509 PKI
- Root-CA and Intermediate CA
- JOSE
- Internationalization
- PAM authentication
- Module extension
